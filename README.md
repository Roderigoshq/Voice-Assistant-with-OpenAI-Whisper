# Voice Assistant with OpenAI Whisper

Uma implementação de assistente de voz em **Python** utilizando o modelo **Whisper** para transcrição de áudio de alta precisão. O projeto foi desenvolvido com foco em modularidade, permitindo a integração de diferentes motores de síntese de voz (TTS) e modelos de linguagem.

### 🚀 Funcionalidades
* **Speech-to-Text (STT):** Transcrição local utilizando o Whisper (OpenAI).
* **Processamento de Intenção:** Estrutura em Python para processar comandos de voz.
* **Resposta em Tempo Real:** Latência otimizada para interações fluidas.
* **Suporte Multilíngue:** Capacidade de processar diferentes idiomas nativamente.
